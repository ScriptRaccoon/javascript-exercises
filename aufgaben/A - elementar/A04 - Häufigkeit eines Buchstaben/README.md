# Häufigkeit eines Buchstaben

## Aufgabe

Implementiere eine Funktion, die die Häufigkeit eines Buchstaben in einem String bestimmt. Wenn der Buchstabe nicht vorkommt, wird `0` zurückgegeben.

## Beispiele

| Eingabe                        | Ausgabe |
| ------------------------------ | ------- |
| `frequency("hallo", "h")`      | `1`     |
| `frequency("hallo", "l")`      | `2`     |
| `frequency("hallo", "o")`      | `1`     |
| `frequency("hallo", "r")`      | `0`     |
| `frequency("javascript", "a")` | `2`     |
| `frequency("javascript", "h")` | `0`     |

## Themen

Schleifen, Strings
