# Palindrome

## Aufgabe

Ein _Palindrom_ ist eine Zeichenkette (_String_), die sich von vorne genauso wie von hinten liest ("abba", "anna", "rentner", usw.).

Implementiere eine Funktion, die bestimmt, ob eine Zeichenkette ein Palindrom ist oder nicht. Der RÃ¼ckgabewert soll `true` oder `false` sein.

## Beispiele

| Eingabe                      | Ausgabe |
| ---------------------------- | ------- |
| `is_palindrome("")`          | `true`  |
| `is_palindrome("a")`         | `true`  |
| `is_palindrome("ab")`        | `false` |
| `is_palindrome("anna")`      | `true`  |
| `is_palindrome("hannah")`    | `true`  |
| `is_palindrome("rentner")`   | `true`  |
| `is_palindrome("rentnerin")` | `false` |

## Themen

Strings, Schleifen
