# Einrahmen eines Arrays

## Aufgabe

Implementiere eine Funktion, die zu einem Array ein Element _vorne und hinten_ anhängt, es also damit quasi "einrahmt".

Hinweis: Das gegebene Array selbst wird nicht verändert. Die Funktion liefert ein neues Array mit den gewünschten Änderungen.

## Beispiele

| Eingabe               | Ausgabe           |
| --------------------- | ----------------- |
| `frame([5, 6, 7], 1)` | `[1, 5, 6, 7, 1]` |
| `frame([], 2)`        | `[2, 2]`          |

## Themen

Arrays, Array-Methoden
