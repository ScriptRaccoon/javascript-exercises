# Aufgabe

Implementiere eine Funktion, die ein Array entgegennimmt und alle Verschiebungen
(shifts) zurückgibt. Das Ergebnis ist also ein Array von Arrays.

# Beispiel

```js
shifts([2, 3, 5]) ==
	[
		[2, 3, 5],
		[3, 5, 2],
		[5, 2, 3],
	];
```

# Themen

Schleifen, Arrays, Einstieg, Modulo-Rechnen, Randfälle
