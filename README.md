# Sammlung von JavaScript-Aufgaben

Dies ist eine Sammlung von JavaScript-Aufgaben mit passenden Lösungen.

Die Aufgaben sind in mehrere Schwierigkeitsgrade (A, B, C, D, E, F) eingeteilt - vom einfachen Einstieg bis zu anspruchsvollen Problemen.

Zu jeder Aufgabe gibt es Lösungen in JavaScript und TypeScript, bzw. ab Schwierigkeitsgrad E nur noch in TypeScript.

Jede Aufgabe muss ohne Zuhilfenahme von Paketen gelöst werden. HTML und CSS kommen nicht vor. Hier wird JavaScript bzw. TypeScript als eine eigenständige Programmiersprache gesehen.
